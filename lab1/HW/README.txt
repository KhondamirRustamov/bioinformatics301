Ваше домашнее задание:

В данной папке 20 .txt файлов в каждом из которых по 20 ID генов (длина последовательности <= 450 bp) из базы данных NCBI.

1) Войдите в базу NCBI (https://www.ncbi.nlm.nih.gov/nuccore/)
2) Найдите последовательность нуклеотидов заданных генов, (введите в поле для поиска все ID через запятую)
3) Скачайте последовательность аминокислот найденных генов:
  3.1) Выберите ВСЕ гены
  3.2) Нажмите Send to -> Complete record -> File -> Format = Fasta -> Create File
4) Далее для всех генов определите:
  
  1) Длину гена
  2) Количество Ц+Г в гене (если нет - 0)
  3) Количество старт кодонов в РНК транскрибированной с данного гена
  4) Количество аминокислот в белке, синтезированного только до первого стоп кодона
  5) Количество Пролинов (P) + Триптофанов (W) в синтезированном белке (если нет - 0)
  
5) Полученные данные запишите в таблицу в EXCEL (если не умеете, можно и в WORD, хотя это НЕ ПРИВЕТСВУЕТСЯ) - у вас должна получиться таблица 20*5, 
подпишите свою фамилию и номер задания и отправьте на почту x.rustamov@cat-science.uz
